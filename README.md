# AI Business Intelligence Platform

An AI-powered Business Intelligence platform that allows users to upload CSV/Excel datasets, analyze business data, and visualize insights through interactive dashboards.

## Current Progress

✅ Frontend completed
✅ Backend APIs developed
✅ Frontend and Backend connected
🔄 Working on file upload functionality
🔄 AI-generated business insights

## Features

- Upload CSV/Excel datasets
- Interactive Dashboard
- KPI Cards
- Charts and Graphs
- Data Cleaning
- AI Business Insights
- Responsive UI
